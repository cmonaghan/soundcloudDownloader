# Soundcloud Downloader

### Here's how to use this tool:
  1. Run `npm install`
  2. Copy the url of a song on soundcloud that you'd like to download
  3. From the command line, type `node soundcloudDownloader.js '[desired song url]'`
  4. Profit!
